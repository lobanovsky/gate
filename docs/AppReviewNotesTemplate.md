# App Review Notes Template

## Purpose

Приложение предназначено для управления дворовыми шлагбаумами и воротами паркинга для авторизованных пользователей.

## How To Review

1. Войти под тестовой учётной записью.
2. На главном экране выбрать одну из зон:
   - `Паркинг`
   - `Двор`
3. Нажать `Заехать` или `Выехать`.
4. Альтернативно использовать кнопку телефонного вызова слева от основной кнопки.
